# 环境配置

```
npm install -g mocha
```

# 目录介绍

## api\routes

api 路由

## api\controllers

控制器层

## services\biz

业务逻辑层

## services\constant

常量 , 状态值

## services\data

操作数据库层

## doc

生成 api 文档 : 选择调试 " 调启 DOC" 生成的文档在 /doc/files 中

## job_scheduler

执行计划任务 : 选择调试 " 示例任务 JOB"

## test

单元测试，使用命令 `npm test` 或 `mocha` 运行
